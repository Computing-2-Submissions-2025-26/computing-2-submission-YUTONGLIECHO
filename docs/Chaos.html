Chaos

Namespace

Chaos

Description:

  • Chaos Gomoku: five-in-a-row on a 15x15 board, with chaotic special skills.

    This module is the authoritative, render-agnostic game engine. Game state is a plain, immutable data object; every function that changes the game returns a brand new state and never mutates its argument. The web app holds the current state and talks to the game only through this API.

    The two players are Chaos.player_one (black, "X", moves first) and Chaos.player_two (white, "O"). A turn is: place one stone, then optionally use one ready skill, then the turn passes to the opponent.

(static) apply(action, state, rngopt) → {object}

Description:

  • Validate and apply an action, returning a result. On success the result holds a brand new Chaos.State and an ordered list of events the view can use for feedback. On an illegal action it returns { ok: false, error } and the original state is untouched (it never mutates or throws).
