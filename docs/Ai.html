Ai

Namespace

Ai

Description:

  • A small, pure computer opponent for Chaos Gomoku.

    The AI only reads the public Chaos API; it never touches game internals. Every choice is a pure function of the state and an injected random source, so the same inputs always produce the same move (tests inject a seeded Rng.seeded). It decides a placement first; the web app then asks for an optional skill and ends the turn.
